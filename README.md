# project

## SETUP

### Prerequisites

- python
- nodejs


### Steps

#### Setup server
1. `cd backend`
2. `pip install -r requirements.txt` (or set up a venv and then do this)
3. `flask --app server run`

Keep the server running in a separate terminal

#### Setup frontend
1. `npm install`
2. `npm run dev`

Access via `http://localhost:5173/`
